Website Guide

Explore the store with all it's features!

Expolore packages, download them and use them. If you really like the package, you can always add it to your favourite package list.

Before downloading the package read about it in the overview of the package. You should always check the system requirements before downloading to see if you have a good enough computer to use the package.

Don't like the package? Or maybe you really like it! Both are great for feedback, so use the review option to leave your opinion on every package you download. It gives other users a better view if the package is worth downloading

Don't like the new update of the package? Or maybe you want to switch to a newer version? Both is possible with the table of versions each package with more than 1 version has.

Have a product ready to deploy? Add it to our store!

To become a content creator just go to your profile page and request it. After that you will be able to upload packages and get the money from each purchase.

Every content creator has authorization to manage his/her packages in the manage packages route. Here you can add, update and delete his/her own packages.
